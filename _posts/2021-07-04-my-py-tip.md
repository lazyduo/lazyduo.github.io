코딩 문제 풀이 시 제가 자주 사용하는 기법을 정리해 보았습니다.

코딩 문제 풀이와 실질적인 프로그래밍은 약간 차이가 있습니다. 그래서 잘 쓰이지 않을 때도 있지만, 코딩 문제 풀이 위해서 꼭 알아두면 좋습니다.

## Reversed List

List slice를 사용하여 '[::-1]' 을 사용하면 reverse 된다.

```python
L1 = [1, 2, 3, 4, 5]
print(L1[::-1])
# 출력 : [5, 4, 3, 2, 1]
```
